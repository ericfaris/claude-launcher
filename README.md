# claude-launcher

If you keep all your coding projects in one folder and use Claude Code as your AI pair programmer, this tool saves you a few steps every time you sit down to work. Instead of manually `cd`-ing into a project and typing `claude`, you just run `lc` and press a single key — the launcher drops you into the right directory and opens Claude automatically.

Technically: a zsh function that presents a styled, numbered menu of your projects directory and launches [Claude Code](https://claude.ai/code) in the selected one. Each project is automatically assigned a [Nerd Font](https://www.nerdfonts.com/) icon by Claude AI on first use, then cached locally. The root folder defaults to `~/projects` and is configurable via `LC_PROJECTS_DIR`.

## Requirements

### 1. zsh

The script uses zsh-specific features (`read -rk 1`, `local -a`, zsh arrays). It must be sourced from `~/.zshrc` and run in a zsh session.

### 2. Claude Code CLI

[Claude Code](https://docs.anthropic.com/en/docs/claude-code) must be installed and the `claude` binary must be on your `$PATH`.

- Install: `npm install -g @anthropic-ai/claude-code`
- Verify: `claude --version`

The launcher calls `claude -p` (print mode) once per new project to generate an icon, and then launches `claude` interactively in the selected directory.

### 3. Nerd Fonts

Icons in the menu are [Nerd Font](https://www.nerdfonts.com/) characters. Your terminal emulator must use a Nerd Font (or a font with Nerd Font patches) for them to render correctly. Without a Nerd Font, you will see boxes, question marks, or missing glyphs instead of icons.

**Installing a Nerd Font:**

1. Download a patched font from [nerdfonts.com](https://www.nerdfonts.com/font-downloads) (e.g. JetBrainsMono Nerd Font, FiraCode Nerd Font, Hack Nerd Font).
2. Install the font on your system:
   - **Linux:** Copy `.ttf`/`.otf` files to `~/.local/share/fonts/` then run `fc-cache -fv`
   - **macOS:** Open the font files with Font Book and click Install
   - **Windows/WSL:** Install the font in Windows, then set your terminal (Windows Terminal, etc.) to use it
3. Set your terminal emulator to use the installed Nerd Font.

**Verifying Nerd Font support:**

Run this in your terminal. If you see a folder icon (), your font is working:

```sh
echo $'\uf07b'
```

### 4. A projects root directory

The launcher scans one level deep inside a root folder and lists each subdirectory as a menu option. It defaults to `~/projects` and can be changed with the `LC_PROJECTS_DIR` environment variable.

```
~/projects/          ← LC_PROJECTS_DIR (default)
├── my-app/
├── dotfiles/
└── scripts/
```

## Installation

1. Clone the repo:
   ```sh
   git clone git@github.com:ericfaris/claude-launcher.git ~/projects/claude-launcher
   ```

2. Source the script in your `~/.zshrc`:
   ```sh
   source ~/projects/claude-launcher/claude-launcher.sh
   ```

3. Reload your shell:
   ```sh
   source ~/.zshrc
   ```

4. Optionally, set a custom root folder (defaults to `~/projects`):
   ```sh
   # Add to ~/.zshrc
   export LC_PROJECTS_DIR="$HOME/dev"
   ```

5. Optionally, auto-launch the menu when you open a shell in the root folder:
   ```sh
   # Add to ~/.zshrc
   [[ "$(pwd)" == "${LC_PROJECTS_DIR:-$HOME/projects}" ]] && lc
   ```

## Usage

Run `lc` to open the menu. Press a single key to select:

| Key | Action |
|-----|--------|
| `1` | `cd` to the root folder (no Claude) |
| `2` | `cd` to the root folder and launch Claude |
| `3`+ | `cd` to that project directory and launch Claude |

## Icon caching

On first use, each project's icon is generated by calling `claude -p` with the project name and a listing of its top-level files. The result is cached in `~/.claude-launcher-icons` (a plain `key=value` text file) so subsequent launches are instant and require no API call.

To reset an icon, delete its line from `~/.claude-launcher-icons` or remove the file entirely:

```sh
rm ~/.claude-launcher-icons
```

If the launcher is invoked from within an existing Claude Code session (detected via `$CLAUDECODE`), it skips the API call and uses the fallback folder icon to avoid nested API calls.
